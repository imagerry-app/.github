# Imagerry

**Private Image Customizer & Converter**  
Edit images, create code mockups, and design OG cards — **100% in your browser**. No uploads. No tracking. Full privacy.

![Imagerry](https://imagerry.com/og.png)

Turn screenshots, logos, and code snippets into polished visuals instantly — with full privacy.

## Highlights
- Image customizer (crop, filters, shadows, watermarks, backgrounds)
- Beautiful code snippet renderer (macOS terminal style + syntax highlighting)
- Open Graph / social card builder with multiple templates
- App icon pack generator (Web, Android, iOS)
- Many other modes that runs locally
- Batch export + high-quality upscaling (up to 8× Lanczos3)
- Full format support: PNG, JPEG, WebP, AVIF, etc.
- PWA + native Desktop apps (Mac/Windows/Linux)

**Privacy-first**: All processing happens locally in your browser. No data leaves your device.
